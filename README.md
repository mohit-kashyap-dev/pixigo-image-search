# 🌐 Pixigo - Image Search App

A responsive image search application built with React.js, integrating REST APIs for real-time data fetching.

---

## 🚀 Features

- Real-time image search
- API integration using Axios
- Routing with React Router
- Image save using localStorage

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Axios
- REST APIs
- Vite
